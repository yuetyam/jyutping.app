偏好設定

使用粵拼輸入法時，右鍵點擊系統右下角的 中 字，即會顯示設定介面。
可選擇字形標準、每頁候選詞數目、候選詞字號大小等選項。



字體

粵拼輸入法會用到很多不太常見的字詞，而電腦自帶字體可能顯示不全或不佳。
推薦安裝以下 全部 幾款字體：
Inter
尙古黑體
遍黑體
MiSans L3



快捷鍵（快速鍵／熱鍵）
快捷鍵 模式以及字形
Shift 切換中文／ABC 模式。
Shift + Space 切換全寬／半寬數字、字母。
Ctrl + . 切換中文／英文標點符號。
Ctrl + Shift + 1 切換至傳統漢字。
Ctrl + Shift + 2 切換至傳統漢字 • 香港標準。
Ctrl + Shift + 3 切換至傳統漢字 • 臺灣標準。
Ctrl + Shift + 4 切換至簡化字。
快捷鍵 候選詞以及輸入
Arrow Up ↑ 或 Shift + Tab 移至上一個候選詞。
Arrow Down ↓ 或 Tab 移至下一個候選詞。
- 或 [ 或 Arrow Left ← 上一頁候選詞。
= 或 ] 或 Arrow Right → 下一頁候選詞。
Esc 取消當前候選輸入。
Ctrl + Shift + Backspace 從輸入記憶移除當前候選詞。


反查粵拼
反查前綴鍵 反查方式
r 普通話拼音反查粵拼。例如: rlin → 林。
v 倉頡五代反查粵拼。例如: vdd → 林。
x 筆畫輸入反查粵拼。例如: xwsad → 木。
q 拆字輸入反查粵拼。例如: qmukmuk → 林。


聲調輸入
聲調映射 舉例
v ⇒ 1 fanv → fan1 芬
x ⇒ 2 fanx → fan2 粉
q ⇒ 3 fanq → fan3 訓
vv ⇒ 4 fanvv → fan4 焚
xx ⇒ 5 fanxx → fan5 奮
qq ⇒ 6 fanqq → fan6 份


卸載

雙擊 C:\Program Files\Jyutping\unins000.exe 即開始卸載。

如果卸載過程中，彈出提示說無法關閉某個程式，可以嘗試：
先將輸入法切換到系統輸入法，再登出、登入一次電腦（或者重啟），然後再開始卸載。
