常見問題（FAQ）


iPhone、iPad 如何使用外置鍵盤？

由於 iOS／iPadOS 系統限制，所有第三方輸入法都無法使用外接鍵盤。
無論是 Magic Keyboard、Smart Keyboard，還是 Magic Keyboard Folio、Smart Keyboard Folio，或者其他外接鍵盤。


iPhone 鍵盤沒有振動／觸覺回饋？

由於 iOS 系統限制，鍵盤需要 允許完全訪問 權限，先可以提供振動（觸感）回應。

去系統設定App 開啟 允許完全訪問 權限給鍵盤之後。
還需要按鍵盤上方的齒輪按鈕，進入設定頁面，選擇啟用振動（觸感）回應。


如何重啟 iPhone、iPad 鍵盤？

有時鍵盤出現問題，需要將它徹底重新啟動。
到系統設定 App，將鍵盤的 允許完全訪問 權限關閉、再開啟。
系統就會將鍵盤重啟。


iPhone、iPad 應用程式文字轉語音朗讀（TTS）沒有聲音？

有幾點需要注意：

  1. 應用程式使用的是系統內建提供的文字轉語音（TTS）功能。
  2. 如果裝置處於靜音狀態，文字轉語音可能會沒有聲音。
  3. 爲保證發音質素，請前往 設定 → 通用 → 語言與地區，添加 繁體中文(香港) 語言。
  4. 爲提高發音質素，請前往 設定 → 輔助功能 → 旁白 → 語音，添加 繁體中文(香港) 語音。
  5. 如果是偶爾沒有聲音，可以嘗試將 粵拼 應用 徹底關閉 後再試。
  6. 或者試一下 重啟一下裝置。
  7. 或者嘗試將系統語言切換成 繁體中文(香港)。
  8. 或者嘗試將 粵拼 應用卸載，然後重新安裝。

macOS 如何輸入簡化字？

用輸入法時按 Control ⌃ + Shift ⇧ + `（位於 esc 鍵下方），會顯示一個選項頁面。
就可以選擇簡化字、標點符號樣式等各種選項。


簡化字模式沒有「是」字？

根據中華人民共和國國務院公布的《通用規範漢字表》，簡化字（簡體字／規範字）只存在「系」一種寫法。


「愛」字讀音是什麼？為什麼粵語歌大多讀 ngoi？

「愛」是清音影母字，當代粵語應當讀零聲母讀音 oi3
「外」是濁音疑母字，當代粵語應當讀疑母讀音 ngoi6

現實中廣州、香港等珠三角各地常有疑影不分，例如 我ngo5 讀成 o5 ， 安on1 讀成 ngon1

理論上可依照粵語聲調區分疑影，零聲母通常搭配陰調（1、2、3），疑母 ng- 通常搭配陽調（4、5、6）

粵語歌中的「愛」大多讀 ngoi3，是因為香港影音娛樂行業推崇的「正音」其實不正，混有部分其他地方的發音偏好，也可能存在對懶音的矯枉過正。


為什麼 macOS 輸入法用 .pkg 來安裝？

  • 第三方輸入法無法上架 Mac App Store。
  • 輸入法安裝需要詢問使用者授權，以安裝於 /Library/Input Methods/ 目錄。
  • 安裝輸入法需要執行輸入法程式自身來做各種操作，還需要執行一系列腳本（script）。


macOS 粵拼輸入法更新失敗？

由於 macOS 對輸入法的管理情況比較複雜，粵拼輸入法內建的「檢查更新／自動更新」有可能會更新失敗。
最簡單、穩定、高效的更新方式，是直接到官網下載安裝最新版。

iPhone、iPad 粵拼輸入法後台活動用電顯著？

粵拼輸入法沒有任何後台活動（Background Activity）。
可能是系統將使用鍵盤的情況當作了後台活動。

是因為你的裝置缺少可以顯示對應字符（字元）的字體。

如果是 iPhone、iPad 或 Android 裝置，將系統更新至最新版本可能會有一些改善。

如果是 macOS，可以安裝大字符集字體來解決。
