Homebrew 安裝 macOS 粵拼輸入法


安裝

 
brew install --cask jyutping
兼容系統： macOS 13 Ventura 或者更高

安裝之後，記得要登出一下電腦再登入，或者重啟一下電腦，以使輸入法正常生效。
請注意：登出電腦會將所有程式結束運行。

安裝並且登出、登入電腦之後，如果沒有看到可用的粵拼輸入法，可以嘗試手動添加：

  1. 前往 系統設定App → 鍵盤 → 文字輸入 → 輸入方式，點擊右邊個 編輯。
  2. 會彈出一個輸入方式列表頁面，點擊列表頁面左下角個 + 號，會再彈出一個語言列表頁面。
  3. 可以在語言頁面左下角輸入「粵拼／粤拼／Jyutping」搜尋，也可以瀏覽尋找語言項 粵語／廣東話／Cantonese。
  4. 請注意，是「粵語／廣東話／Cantonese」，而不是「繁體粵語／繁體廣東話／Cantonese, Traditional」。
  5. 找到 粵拼／Jyutping，選中它，再點擊頁面右下角 加入。



更新

 

brew upgrade --cask jyutping

更新之後，最好和樣登出一下電腦再登入，或者重啟一下電腦，以使輸入法正常生效。
請注意：登出電腦會將所有程式結束運行。


卸載

如果想要卸載輸入法，可以用以下兩個命令之一。

1. 卸載輸入法並刪除相關資料： 

brew uninstall --zap --cask jyutping

2. 只卸載輸入法，保留相關資料： 

brew uninstall --cask jyutping

卸載完之後，記得和樣要登出一下電腦再登入，或者重啟一下電腦。
請注意：登出電腦會將所有程式結束運行。
