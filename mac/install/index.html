macOS 粵拼輸入法下載安裝指引


系統要求： macOS 12 Monterey 或者更高

下載 • Download

Jyutping-v0.25.0-Mac-IME.pkg

SHA-256: 

d795a003dae7a781f2702a6e3d05cd727be643ce47e7a0d6940daff5a7deb95d


安裝 • Install

雙擊下載到嘅 pkg 檔案，按步驟完成安裝。
到最後一步，安裝器會請求你登出電腦再登入，以使輸入法正常生效。

安裝之後，如果見毋到有粵拼輸入法，請前往 系統設定 → 鍵盤 → 輸入方式 度，手動添加。
添加箇陣，「粵拼」位於 Cantonese/粵語/廣東話 語言項，可以用 Jyutping/粵拼/粤拼 來搜尋搵。
（請注意係 Cantonese/粵語/廣東話，而毋係 Cantonese, Traditional/繁體粵語/繁體廣東話。）


卸載 • Uninstall

如果你需要卸載本輸入法：

去 系統設定 → 鍵盤 → 輸入方式 度，移除「粵拼/Jyutping」。

刪除以下檔案／檔案夾： 

/Library/Input\ Methods/Jyutping.app
~/Library/Application\ Scripts/org.jyutping.inputmethod.Jyutping
~/Library/Containers/org.jyutping.inputmethod.Jyutping
~/Library/Preferences/org.jyutping.inputmethod.Jyutping.plist

最後，登出電腦再登入，或者重啓電腦。
