Homebrew 安裝 macOS 粵拼輸入法


兼容系統： macOS 13 Ventura 或者更高

安裝 • Install 

brew install --cask jyutping

安裝之後，記得要登出一下電腦再登入，或者重啓一下電腦，以使輸入法正常生效。
請注意：登出電腦會將所有程式結束運行。

安裝並且登出、登入電腦之後，如果冇見到有粵拼輸入法可用，可以試下手動添加：

  1. 前往 系統設定App → 鍵盤 → 文字輸入 → 輸入方式，點擊右邊個「編輯」。
  2. 會彈出一個輸入方式列表䈎面，點擊列表䈎面左下角個 + 號，會再彈出一個語言列表䈎面。
  3. 可以喺語言䈎面左下角輸入「粵拼／粤拼／Jyutping」搜尋，亦可以瀏覽搵語言項「粵語／廣東話／Cantonese」。
  4. 請注意，係「粵語／廣東話／Cantonese」，而非「繁體粵語／繁體廣東話／Cantonese, Traditional」。
  5. 搵到「粵拼／Jyutping」，選中佢，再點擊䈎面右下角「添加」。



更新 • Upgrade 

brew upgrade --cask jyutping

更新之後，最好同樣登出一下電腦再登入，或者重啓一下電腦，以使輸入法正常生效。
請注意：登出電腦會將所有程式結束運行。


卸載 • Uninstall

如果想要卸載輸入法，可以用以下兩個命令之一。

1. 卸載輸入法並刪除相關資料： 

brew uninstall --zap --cask jyutping

2. 只卸載輸入法，保留相關資料： 

brew uninstall --cask jyutping

卸載完之後，記得同樣要登出一下電腦再登入，或者重啓一下電腦。
請注意：登出電腦會將所有程式結束運行。
