Homebrew 安裝 macOS 粵拼輸入法


兼容系統： macOS 13 Ventura 或者更高

安裝 • Install 

brew install --cask jyutping

安裝之後，記得要登出一下電腦再登入，或者重啓一下電腦，以使輸入法正常生效。
（登出電腦會將所有程式結束運行。）

如果輸入法列表見毋到有粵拼輸入法，請前往 系統設定 → 鍵盤 → 輸入方式，手動添加。
添加嗰陣，「粵拼」位於 Cantonese/粵語/廣東話 語言項，亦可以用 粵拼/粤拼/Jyutping 來搜尋搵。
（請注意係 Cantonese/粵語/廣東話，而毋係 Cantonese, Traditional/繁體粵語/繁體廣東話。）


更新 • Upgrade 

brew upgrade --cask jyutping

更新之後，最好同樣登出一下電腦再登入，或者重啓一下電腦，以使輸入法正常生效。
（登出電腦會將所有程式結束運行。）


卸載 • Uninstall

如果想要卸載輸入法，可以用以下兩個命令之一。

1. 卸載輸入法並刪除相關檔案（文件）／檔案夾（文件夾）： 

brew uninstall --zap --cask jyutping

2. 只卸載輸入法，毋刪除相關檔案（文件）／檔案夾（文件夾）： 

brew uninstall --cask jyutping

卸載完之後，記得同樣要登出一下電腦再登入，或者重啓一下電腦。
（登出電腦會將所有程式結束運行。）
