安裝 • Install

安裝步驟：

  1. 雙擊下載到嘅 pkg 檔案（文件），開始安裝。
  2. 中途系統可能會要你輸入電腦密碼以授權安裝。
  3. 中途系統設定App可能會彈出來，提示添加「粵拼」輸入法。
  4. 到最後一步，安裝程式會請示登出電腦，以使輸入法正常生效。
請注意：登出電腦會將所有程式結束運行。

安裝之後，如果見毋到有粵拼輸入法，請前往 系統設定 → 鍵盤 → 輸入方式，手動添加。
添加嗰陣，「粵拼」位於 粵語/廣東話/Cantonese 語言項，亦可以用「粵拼/粤拼/Jyutping」來搜尋搵。
（請注意係 粵語/廣東話/Cantonese，而毋係 繁體粵語/繁體廣東話/Cantonese, Traditional。）


更新 • Update

由於 macOS 對輸入法嘅管理情況比較複雜，粵拼輸入法內置嘅「檢查更新／自動更新」有可能會更新失敗。
所以更推薦來官網下載安裝最新版，或者 通過 Homebrew 安裝／更新 macOS 粵拼輸入法。
