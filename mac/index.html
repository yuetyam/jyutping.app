安裝 • Install

安裝步驟：

  1. 雙擊下載到嘅 pkg 文件（檔案），開始安裝。
  2. 中途系統可能會要你輸入電腦密碼授權系統進行安裝。
  3. 中途系統設定App可能會彈出來，請示添加「粵拼」輸入法。
  4. 到最後一步，安裝程式會請示登出電腦，以使輸入法正常生效。
請注意：登出電腦會將所有程式結束運行。

安裝並且登出、登入電腦之後，如果冇見到有粵拼輸入法可用，可以試下手動添加：

  1. 前往 系統設定App → 鍵盤 → 文字輸入 → 輸入方式，點擊右邊個「編輯」。
  2. 會彈出一個輸入方式列表䈎面，點擊列表䈎面左下角個 + 號，會再彈出一個語言列表䈎面。
  3. 可以喺語言䈎面左下角輸入「粵拼／粤拼／Jyutping」搜尋，亦可以瀏覽搵語言項「粵語／廣東話／Cantonese」。
  4. 請注意，係「粵語／廣東話／Cantonese」，而非「繁體粵語／繁體廣東話／Cantonese, Traditional」。
  5. 搵到「粵拼／Jyutping」，選中佢，再點擊䈎面右下角「添加」。



更新 • Update

由於 macOS 對輸入法嘅管理比較複雜，粵拼輸入法內置嘅「檢查更新／自動更新」有可能會更新失敗。
所以更推薦直接來官網下載安裝最新版，或者用 Homebrew 安裝／更新 macOS 粵拼輸入法。
