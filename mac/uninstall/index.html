如何卸載 macOS 粵拼輸入法


如果你嘅輸入法係用 Homebrew 安裝嘅，卸載請參攷: Homebrew 安裝 macOS 粵拼輸入法

以下係手動卸載(Uninstall / 移除) macOS 粵拼輸入法步驟：

首先，前往 系統設定App → 鍵盤 → 輸入方式，移除「粵拼／Jyutping」。

然後，刪除輸入法應用程式本體：

  1. 開啓 Finder 再撳鍵盤組合鍵 Command ⌘ + Shift ⇧ + G 會彈出一個䈎面
  2. 輸入路徑 /Library/Input Methods/ 再撳鍵盤 Return ↵ 鍵前往
  3. 搵到 Jyutping.app 將佢刪除。

如果刪除 Jyutping.app 時，彈提示話佢運行緊、無法刪除。需要先將佢結束運行：
方法一：去 Activity Monior(活動監視器) 搵到「粵拼／Jyutping」，將佢結束運行。
方法二：去 Terminal(終端) 運行以下命令將佢結束運行： 

osascript -e 'tell application id "org.jyutping.inputmethod.Jyutping" to quit'


如果你想刪晒粵拼輸入法相關嘅所有記錄、設定等，可以刪除以下資料： 

~/Library/Application Scripts/org.jyutping.inputmethod.Jyutping
~/Library/Caches/org.jyutping.inputmethod.Jyutping
~/Library/Containers/org.jyutping.inputmethod.Jyutping

最後，登出電腦再登入，或者重啓電腦。
