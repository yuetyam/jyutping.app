Install

Double-click the downloaded exe file to start installing.
For detailed instructions on adding an input method, see How to add an input method in Windows - ASUS.


Settings

While using Jyutping Input Method, right-click 中 in the bottom-right corner of the system taskbar to open the settings interface.
You can choose the character form standard, the number of candidates per page, the candidate font size, and more.


Fonts

Jyutping Input Method uses many less common characters and words, and the fonts bundled with your computer may not display them completely or well.
We recommend installing all of the following fonts:
Inter
Shanggu Sans
Plangothic
MiSans L3


Shortcuts
Shortcut Mode and character form
Shift Toggle Chinese / ABC mode.
Shift + Space Toggle full-width / half-width numbers and letters.
Ctrl + . Toggle Chinese / English punctuation.
Ctrl + Shift + 1 Switch to Traditional Chinese.
Ctrl + Shift + 2 Switch to Traditional Chinese • Hong Kong standard.
Ctrl + Shift + 3 Switch to Traditional Chinese • Taiwan standard.
Ctrl + Shift + 4 Switch to Simplified Chinese.
Shortcut Candidates and input
Arrow Up ↑ or Shift + Tab Move to the previous candidate.
Arrow Down ↓ or Tab Move to the next candidate.
- or [ or Arrow Left ← Go to the previous candidate page.
= or ] or Arrow Right → Go to the next candidate page.
Esc Cancel the current candidate input.
Ctrl + Shift + Backspace Remove the current candidate from input memory.

Uninstall

Double-click C:\Program Files\Jyutping\unins000.exe to start uninstalling.

If the uninstaller says it cannot close an app during the process, try switching to a system input method first, then sign out and sign in again, or restart the computer, before uninstalling again.
