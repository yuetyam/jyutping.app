Jyutping Keyboard

A Cantonese Jyutping input method that is simple, clear, fast, and accurate.
If you can speak Cantonese, you can type with it naturally.
Every input is a chance to review and reinforce pronunciation. The more you use Jyutping, the more familiar Cantonese becomes.

Install and start typing. No complicated setup is required.
Open source, free, ad-free, fully offline, and designed with privacy in mind.

Built on the Linguistic Society of Hong Kong Cantonese Romanization Scheme, with support for many common spelling habits.
You can also look up Jyutping with Mandarin Pinyin, Cangjie, Sucheng, strokes, and character decomposition.
