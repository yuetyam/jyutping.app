Install Jyutping Input Method for macOS with Homebrew


Install

 
brew install --cask jyutping
Compatibility: macOS 13 Ventura or later

After installation, log out and log back in, or restart your computer, so the input method can take effect.
Note: logging out closes all running apps.

After installation and after logging out and back in, if Jyutping is not available, try adding it manually:

  1. Go to System Settings → Keyboard → Text Input → Input Sources, then click Edit on the right.
  2. In the input source list, click the + button at the lower left to open the language list.
  3. Search for “粵拼 / 粤拼 / Jyutping” at the lower left, or browse for Cantonese.
  4. Use Cantonese, not Cantonese, Traditional.
  5. Select 粵拼 / Jyutping, then click Add at the lower right.



Update

 

brew upgrade --cask jyutping

After updating, it is also best to log out and log back in, or restart your computer, so the input method can take effect.
Note: logging out closes all running apps.


Uninstall

To uninstall the input method, use one of the following two commands.

1. Uninstall the input method and delete related data: 

brew uninstall --zap --cask jyutping

2. Uninstall the input method and keep related data: 

brew uninstall --cask jyutping

After uninstalling, also log out and log back in, or restart your computer.
Note: logging out closes all running apps.
