Jyutping Input Method for macOS

Using Homebrew? You can install Jyutping Input Method for macOS with Homebrew.


Download

Download: Jyutping-v0.77.0-Mac.pkg
Compatibility: macOS 13 Ventura or later
CPU support: Apple Silicon and Intel x86_64

Install

Installation steps:

  1. Double-click the downloaded pkg file to start installation.
  2. macOS may ask for your computer password to authorize installation.
  3. System Settings may open and ask to add the Jyutping input method.
  4. At the last step, the installer asks you to log out so the input method can take effect.
Note: logging out closes all running apps.

After installation and after logging out and back in, if Jyutping is not available, try adding it manually:

  1. Go to System Settings → Keyboard → Text Input → Input Sources, then click Edit on the right.
  2. In the input source list, click the + button at the lower left to open the language list.
  3. Search for “粵拼 / 粤拼 / Jyutping” at the lower left, or browse for Cantonese.
  4. Use Cantonese, not Cantonese, Traditional.
  5. Select 粵拼 / Jyutping, then click Add at the lower right.



Update

Because macOS manages input methods in a fairly complex way, the built-in “Check for Updates / Automatic Updates” feature may fail.
We recommend downloading and installing the latest version from this website, or installing/updating Jyutping Input Method for macOS with Homebrew.

How do I type simplified Chinese?

While using the input method, press Control ⌃ + Shift ⇧ + ` (below the esc key). An options panel will appear where you can choose simplified Chinese, punctuation style, and other options.
