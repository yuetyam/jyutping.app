常問問題（FAQ）


點解 iPadOS 用毋到外置鍵盤？

由於系統限制，所有第三方輸入法都冇辦法使用外置鍵盤。冇計。


點解 iPhone 打字冇振動？

由於系統限制，鍵盤需要「允許全權取用」權限，先可以提供振動反饋。

去系統設定App 開啓「允許全權取用」權限畀邇個鍵盤之後。
仲需要撳邇個鍵盤上部個齒輪掣，入到設定䈎面，去開啓「按鍵振動反饋」。


點解 macOS 有時打毋出字？

如果你係啱啱安裝／更新咗輸入法，請確保登出再登入咗一下電腦，以使輸入法正常生效。

亦有可能係輸入法自己嘅問題，請嘗試：
切換到其他輸入法，再切換返來。
換一個地方／窗口打字，再換返來。

如果再毋得，可能要將邇個輸入法結束運行。
方法一：去 Activity Monior 搵到「粵拼/Jyutping」，將佢結束。
方法二：去 Terminal 運行以下邇一行命令： 

osascript -e 'tell application id "org.jyutping.inputmethod.Jyutping" to quit'
方法三：登出電腦再登入，或者重啓電腦。


點解某啲候選詞會顯示成問號方塊？

係因爲你嘅設備缺少可以顯示邇啲字符嘅字體。

如果你嘅係 iPhone 或 iPad，將系統更新至最新可能會有少少改善。

如果你嘅係 macOS，可以通過安裝大字符集嘅字體來解決。
推薦安裝以下全部四款字體：

秋空黑體　（直接下載： ChiuKongGothic-CL-Regular.otf.xz）
遍黑體　　（直接下載： Plangothic.xz）
一點明體　（直接下載： I.Ming.ttf.xz）
花園明朝體（直接下載： HanaMinB.ttf.xz）

如果你毋知點樣安裝字體，可以睇下邇篇 Apple 支援。
