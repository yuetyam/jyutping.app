常問問題（FAQ）


iPhone、iPad 如何使用外置鍵盤？

由於 iOS／iPadOS 系統限制，所有第三方輸入法都冇辦法使用外置鍵盤。
無論係 Magic Keyboard、Smart Keyboard，定係 Magic Keyboard Folio、Smart Keyboard Folio，抑或其他外置鍵盤。


iPhone 鍵盤冇振動／觸感回應？

由於 iOS 系統限制，鍵盤需要 允許完全訪問 權限，先可以提供振動（觸感）回應。

去系統設置App 開啓 允許完全訪問 權限畀鍵盤之後。
仲需要撳鍵盤上部個齒輪掣，入到設置䈎面，去選擇啓用振動（觸感）回應。


如何重啓 iPhone、iPad 鍵盤？

有時鍵盤出現問題，需要將佢徹底重啓。
去系統設置App，將鍵盤嘅 允許完全訪問 權限關閉、再開啓。
系統就會將鍵盤重啓。


iPhone、iPad 應用程式文字轉語音朗讀（TTS）冇聲音？

有幾點需要注意：

  1. 應用程式使用嘅係，系統內置提供嘅文字轉語音（TTS）功能。
  2. 如果設備處於靜音狀態，文字轉語音可能會冇聲音。
  3. 爲保證發音質素，請前往 設置 → 通用 → 語言與地區，添加 繁體中文(香港) 語言。
  4. 爲提高發音質素，請前往 設置 → 輔助功能 → 旁白 → 語音，添加 繁體中文(香港) 語音。
  5. 如果係偶爾冇聲音，可以試下將 粵拼 應用 徹底關閉 再試。
  6. 或者試下 重啓一下設備。
  7. 或者試下將系統語言切換成 繁體中文(香港)。
  8. 或者試下將 粵拼 應用卸載，然後再重新安裝。

macOS 如何輸入簡化字？

用輸入法時撳 Control ⌃ + Shift ⇧ + `（位於 esc 鍵下方），會顯示一個選項䈎面。
就可以揀簡化字、標點符號樣式等各種選項。


簡化字模式冇「係」字？

根據中華人民共和國國務院公佈嘅《通用規範漢字表》，簡化字（簡體字／規範字）只存在「系」一種寫法。


「愛」字讀音係乜？點解粵語歌多數讀 ngoi ？

「愛」係清音影母字，當代粵語應當讀零聲母讀音 oi3
「外」係濁音疑母字，當代粵語應當讀疑母讀音 ngoi6

現實中廣州、香港等珠三角各地常有疑影不分，例如 我ngo5 讀成 o5 ， 安on1 讀成 ngon1

理論上可依照粵語聲調區分疑影，零聲母通常搭配陰調（1、2、3），疑母 ng- 通常搭配陽調（4、5、6）

粵語歌「愛」多數讀 ngoi3 ， 係因爲香港影音娛樂行業推崇嘅「正音」其實不正，混有部分其他地方嘅發音偏好，又或存在對懶音嘅矯枉過正。


點解 macOS 輸入法用 .pkg 來安裝？

  • 第三方輸入法無法上架 Mac App Store。
  • 輸入法安裝需要詢問用戶授權以安裝於 /Library/Input Methods/ 目錄。
  • 安裝輸入法需要運行輸入法程式自身來做各種操作，仲需要運行一系列腳本(script)。


macOS 粵拼輸入法更新失敗？

由於 macOS 對輸入法嘅管理情況比較複雜，粵拼輸入法內置嘅「檢查更新／自動更新」有可能會更新失敗。
最簡單、穩陣、高效嘅更新方式，係直接來官網下載安裝最新版。

iPhone、iPad 粵拼輸入法後臺活動用電顯著？

粵拼輸入法係冇任何後臺活動(Background Activity)嘅。
可能係系統將使用鍵盤嘅情景當作咗係後臺活動。


iPhone、iPad 鍵盤「貼上」掣

請確保：

  • 開啓 允許完全訪問 權限畀粵拼鍵盤。
  • 將 從其他App貼上 設置項選爲允許。


某啲候選詞會顯示成問號、叉號或者一嚿嚿？例如 ￾

係因爲你嘅設備缺少可以顯示對應字符（字元）嘅字體。

如果係 iPhone、iPad 或者 Android，將系統更新至最新可能會有少少改善。

如果係 macOS，可以安裝大字符集嘅字體來解決。
