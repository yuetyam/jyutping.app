常問問題（FAQ）


點解 iPadOS 用毋到外置鍵盤？

由於系統限制，所有第三方輸入法都冇辦法使用外置鍵盤。
無論係 Magic Keyboard、Smart Keyboard，定係 Magic Keyboard Folio、Smart Keyboard Folio，抑或其他外置鍵盤。


點解 iPhone 打字冇振動／觸感回應？

由於系統限制，鍵盤需要「允許全權取用」權限，先可以提供振動／觸感回應。

去系統設定App 開啓「允許全權取用」權限畀邇個鍵盤之後。
仲需要撳邇個鍵盤上部個齒輪掣，入到設定䈎面，去選擇啓用振動／觸感回應。


應用程式嘅文字語音朗讀（TTS）冇聲音？

有幾點需要注意：
1. 應用程式使用嘅係，系統提供嘅文字語音朗讀（TTS）功能。
2. 如果設備處於靜音狀態，文字語音朗讀可能會冇聲音。
3. 爲保證發音質素，請前往 設定 → 一般 → 語言與地區，添加 繁體中文(香港) 語言。
4. 爲提高發音質素，請前往 設定 → 輔助使用 → 旁白 → 語音，添加 繁體中文(香港) 語音。


macOS 點樣輸入簡化字？

撳組合掣 Control + Shift + ` (esc下邊箇粒掣)，會出現一個選項䈎面。
就可以揀簡化字、標點符號樣式等各種選項。


點解 macOS 有時打毋出字？

如果你係啱啱安裝／更新輸入法，請確保登出再登入一下電腦，以使輸入法正常生效。

亦有可能係輸入法自己嘅問題，請嘗試：
• 切換到其他輸入法，再切換返來。
• 換一個地方／視窗打字，再換返來。
• 將輸入對象App結束運行，再重新開啓。

如果再毋得，可能要將邇個輸入法結束運行。
方法一：去 Activity Monior(活動監視器) 搵到「粵拼/Jyutping」，將佢結束運行。
方法二：去 Terminal(終端) 運行以下命令將佢結束運行： 

osascript -e 'tell application id "org.jyutping.inputmethod.Jyutping" to quit'
方法三：登出電腦再登入，或者重啓電腦。


電腦 Google Docs 網頁打字冇候選詞介面？

主要係因爲 Google Docs 係用 Canvas 來繪製䈎面。(source)
包括系統自帶輸入法在內，大部分輸入法都會或多或少無法完美兼容。
惟有期望 Google 整返好佢。
不過好彩，當前有辦法可嘗試緩解邇個問題：

第一步：撳 Google Docs 個 Tools 選單，搵到 Accessibility，撳入去。
第二步：勾選 Turn on screen magnifier support，然後撳 OK 掣。


點解 macOS 輸入法用 .pkg 來安裝？

• 第三方輸入法無法上架 Mac App Store。
• 輸入法需要安裝於 /Library/Input\ Methods/ 目錄，而毋係通常嘅 /Applications/ 目錄。安裝過程中需要詢問用戶授權。
• 安裝輸入法需要運行輸入法程式本身來做各種操作，仲需要運行一系列腳本(script)。


macOS 粵拼輸入法更新失敗？

如果係通過「檢查更新／自動更新」更新失敗，請來官網手動下載安裝最新版。

點解某啲候選詞會顯示成問號方塊？

係因爲你嘅設備缺少可以顯示邇啲字符（字元）嘅字體。

如果係 iPhone 或 iPad，將系統更新至最新可能會有少少改善。

如果係 macOS，可以通過安裝大字符（字元）集字體來解決。
推薦安裝以下 全部 幾款字體：

SF Pro & SF Mono
Monaspace Neon
Inter
尙古黑體
遍黑體
MiSans L3
