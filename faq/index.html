點解 iPadOS 用唔到外置鍵盤？

由於系統限制，所有第三方輸入法都冇辦法使用外置鍵盤。冇計。


點解 macOS 有時打唔出字？

如果你係啱啱安裝／更新咗輸入法，請確保登出再登入咗一下電腦，以使輸入法正常生效。

亦有可能係輸入法自己嘅問題，請嘗試：
切換到其他輸入法，再切返返來。
換一個地方／窗口打字。

如果再唔得，可能要將呢個輸入法結束運行。
方法一：去 Activity Monior 搵到「粵拼/Jyutping」，將佢結束。
方法二：去 Terminal 運行以下呢一行命令：
osascript -e 'tell application id "org.jyutping.inputmethod.Jyutping" to quit'
