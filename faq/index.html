常問問題（FAQ）


點解 iPadOS 用唔到外置鍵盤？

由於系統限制，所有第三方輸入法都冇辦法使用外置鍵盤。冇計。


點解 iPhone 打字冇振動？

由於系統限制，鍵盤需要「允許全權取用」權限，先可以提供振動反饋。

去系統設定App 開啓「允許全權取用」權限畀呢個鍵盤之後。
仲需要撳呢個鍵盤上部個齒輪掣，入到設定䈎面，去開啓「按鍵振動反饋」。


點解 macOS 有時打唔出字？

如果你係啱啱安裝／更新咗輸入法，請確保登出再登入咗一下電腦，以使輸入法正常生效。

亦有可能係輸入法自己嘅問題，請嘗試：
切換到其他輸入法，再切換返來。
換一個地方／窗口打字，再換返來。

如果再唔得，可能要將呢個輸入法結束運行。
方法一：去 Activity Monior 搵到「粵拼/Jyutping」，將佢結束。
方法二：去 Terminal 運行以下呢一行命令：
osascript -e 'tell application id "org.jyutping.inputmethod.Jyutping" to quit'
方法三：登出電腦再登入，或者重啓電腦。
