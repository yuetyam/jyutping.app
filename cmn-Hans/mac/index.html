macOS 粤拼输入法

正在使用 Homebrew？可以用 Homebrew 安装 macOS 粤拼输入法


下载

下载: Jyutping-v0.77.0-Mac.pkg
兼容: macOS 13 Ventura 或者更高
兼容: Apple Silicon 以及 Intel x86_64 CPU

大小（Size）: 26779057 字节(Bytes)

安装

安装步驟：

  1. 双击下载到的 pkg 文件，开始安装。
  2. 安装过程中，系统可能会要求你输入电脑密码授权安装。
  3. 安装过程中，系统设置 App 可能会弹出，提示添加 粤拼 输入法。
  4. 到最后一步，安装程式会提示登出电脑，以使输入法正常生效。
请注意：登出电脑会将所有程式结束运行。

安装并且登出、登入电脑之后，如果没有看到可用的粤拼输入法，可以尝试手动添加：

  1. 前往 系统设置 App → 键盘 → 文字输入 → 输入方式，点击右边个 编辑。
  2. 会弹出一个输入方式列表页面，点击列表页面左下角个 + 号，会再弹出一个语言列表页面。
  3. 可以在语言页面左下角输入「粤拼／粤拼／Jyutping」搜索，也可以浏览寻找语言项 粤语／广东话／Cantonese。
  4. 请注意，是「粤语／广东话／Cantonese」，而不是「繁体粤语／繁体广东话／Cantonese, Traditional」。
  5. 找到 粤拼／Jyutping，选中它，再点击页面右下角 加入。



更新

由于 macOS 对输入法的管理比较复杂，粤拼输入法内建的「检查更新／自动更新」有可能会更新失败。
所以更推荐直接到官网下载安装最新版，或者用 Homebrew 安装／更新 macOS 粤拼输入法。

如何输入简化字？

用输入法时按 Control ⌃ + Shift ⇧ + `（位于 esc 键下方），会显示一个选项页面，可以选择简化字、标点符号样式等各种选项。
