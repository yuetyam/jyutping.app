Homebrew 安装 macOS 粤拼输入法


安装

 
brew install --cask jyutping
兼容系统： macOS 13 Ventura 或者更高

安装之后，记得要登出一下电脑再登入，或者重启一下电脑，以使输入法正常生效。
请注意：登出电脑会将所有程式结束运行。

安装并且登出、登入电脑之后，如果没有看到可用的粤拼输入法，可以尝试手动添加：

  1. 前往 系统设置 App → 键盘 → 文字输入 → 输入方式，点击右边个 编辑。
  2. 会弹出一个输入方式列表页面，点击列表页面左下角个 + 号，会再弹出一个语言列表页面。
  3. 可以在语言页面左下角输入「粤拼／粤拼／Jyutping」搜索，也可以浏览寻找语言项 粤语／广东话／Cantonese。
  4. 请注意，是「粤语／广东话／Cantonese」，而不是「繁体粤语／繁体广东话／Cantonese, Traditional」。
  5. 找到 粤拼／Jyutping，选中它，再点击页面右下角 加入。



更新

 

brew upgrade --cask jyutping

更新之后，最好同样登出一下电脑再登入，或者重启一下电脑，以使输入法正常生效。
请注意：登出电脑会将所有程式结束运行。


卸载

如果想要卸载输入法，可以用以下两个命令之一。

1. 卸载输入法并删除相关资料： 

brew uninstall --zap --cask jyutping

2. 只卸载输入法，保留相关资料： 

brew uninstall --cask jyutping

卸载完之后，记得同样要登出一下电脑再登入，或者重启一下电脑。
请注意：登出电脑会将所有程式结束运行。
