如何卸载 macOS 粤拼输入法

如果你的输入法是用 Homebrew 安装的，卸载请参考: Homebrew 安装 macOS 粤拼输入法

以下是手动卸载（Uninstall / 移除）macOS 粤拼输入法的步驟：

首先，前往 系统设置 App → 键盘 → 文字输入 → 输入方式，移除 粤拼。

然后，删除输入法应用程序本体：

  1. 开启 Finder 再按键盘組合键 Command ⌘ + Shift ⇧ + G 会弹出一个页面
  2. 输入路径 /Library/Input Methods/ 再按键盘 Return ↵ 键前往
  3. 找到 Jyutping.app 并将它删除。

如果删除 Jyutping.app 时，系统提示它正在执行、无法删除，需要先结束它的执行：
方法一：到 Activity Monitor（活动監视器）找到「粤拼／Jyutping」，结束它的执行。
方法二：到 Terminal（终端）执行以下命令结束它的执行： 

osascript -e 'tell application id "org.jyutping.inputmethod.Jyutping" to quit'


如果你想删除粤拼输入法相关的所有记录、设置等，可以删除以下资料： 

~/Library/Application Scripts/org.jyutping.inputmethod.Jyutping
~/Library/Caches/org.jyutping.inputmethod.Jyutping
~/Library/Containers/org.jyutping.inputmethod.Jyutping

最后，登出电脑再登入，或者重启电脑。
